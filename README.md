# TinyLanguageModel
Built a autoregressive language model from scratch in PyTorch, implementing tokenization, embeddings, causal multi-head self-attention, Transformer blocks, training loop, checkpointing, and temperature-based text generation. Trained and evaluated the model on a custom text corpus, tracking training/validation loss and generation quality.
